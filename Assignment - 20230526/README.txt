1. What are Meta Tags? and list of meta tags
Meta tags provides more information of the website and current page.

a. <meta name="description" content="Free Web tutorials for HTML and CSS">
Define description of the webpage

b. <meta name="keywords" content="India, Play ground">
Defines keyword for Search engine.

c. <meta name="viewport" content="width=device-width, initial-scale=1.0">
content="width=device-width - Width of webpage should take width of device
initial-scale=1.0 - Defines the zooming scale of the webpage.

d. <meta http-equiv="X-UA-Compatible" content="IE=edge">
Specifies which version of IE website should be compatible.

e. <meta charset="UTF-8">
Speaks about the character encoding.

2. Web Content Accessibility Guideline? Read about A, AA, AAA.
The Web Content Accessibility Guideline provides a set of guidelines for making web
content more accessible to all people and with a wide variety of disabilities.

1. Level A – Level A success criteria are required for all websites. If website is not compliant with Level A,
it may have severe accessibility concerns that prevent people with disabilities from using it.

2. Level AA – Websites that passes the WCAG Level AA are reasonably accessible for the majority of users.
The majority of websites should strive towards Level AA compliance.
To accomplish this objective, content must achieve all Level AA and Level A success criteria.

3. Level AAA – Web content that conforms to WCAG guidelines Level AAA accessibility is considered ideal.
To achieve Level AAA website must pass the Level A and Level AAA
